## Get Started

Install the dependencies

```bash
yarn install
```

Run the dev build

```bash
yarn dev
```
Navigate to [localhost:5000](http://localhost:5000). 

## Technologies Used
- NodeJS
- SvelteJS
- Google Firebase
  - Google Authentication
  - Firestore
  
## Features
- Lightweight
- Easy-to-use
- Real-time updates
- Secured by Google Auth
- Data storage
